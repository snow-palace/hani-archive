4. 社团系统
摘要： 玩家生存的基础单位。社团的成立需要经过严苛的审核。社团[首席]拥有绝对的权利，同时也承担被挑战的风险。
📝 成立与申请
  • 人数要求：1人以上即可申请。
  • 首席选举：多人社团需投票选出[社团首席]。
  • 据点要求：必须找到一处空置建筑物作为[社团据点]，并指定[挑战房间]。
  • 流程：向[社联组]递交表格 → 7日现场勘查 → 7日审核 → 7日公示。
⚖️ 权利与义务
  • 校徽归属：所有成员获得的[校徽卡]，所属权归[首席]所有。
  • 强制应战：一旦有外人攻入[挑战房间]，首席必须接受挑战（称为[首席战]）。
  • 代理人规则：首席可指定成员作为[代理人]出战。
☠️ 获胜报酬 (赌注)
首席战败： 支付1张[校徽卡] (或2张随机稀有卡)。
代理人战败： 支付翻倍 (2张校徽卡 或 4张稀有卡)。
战死： 胜者获得败者持有的全部斑丘卡和属性块。
5. 战斗种类
摘要： 除了核心的首席战，斑丘中还存在多种冲突形式。请注意，R0之后规则修改为“强制1v1”，禁止第三方干涉。
一般战
野外遭遇，最常见的战斗形式。击杀可获得对方初始属性块。
遭遇战
突发性的短兵相接，通常发生在争夺物资时。
团体战
社团与社团之间的大规模冲突。(注：新规则限制了混战规模)
赌约战
双方约定赌注进行的决斗，受系统规则保护。
首席战
攻入据点核心触发。关于校徽归属的最高级别战斗。
