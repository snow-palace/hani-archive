# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述 / Project Overview

Static HTML website for displaying game/story content - characters, cards, timelines, and rules. The main file `index.html` contains all views and logic (~10K lines).

## 技术栈 / Tech Stack

- Pure static HTML/CSS/JavaScript (single file `index.html`)
- Node.js script for generating card data

## 常用命令 / Common Commands

```bash
# Update card data after adding new card images
node generate_list.js
```

## 目录结构 / Directory Structure

```
images/
  BQcards/cards_front/  # Card images (RW-, JN-, WP-, JK- prefix categories)
  icon/                  # Website icons
cardData.js              # Auto-generated, do not edit manually
generate_list.js         # Card data generation script
index.html               # Main site page
```

## 卡牌系统 / Card System

- **RW** (人物/Characters): 48 cards
- **JN** (技能/Skills): 32 cards
- **WP** (物品/Items): 41 cards
- **JK** (纪念/Mementos): 20 cards
- **Total**: 141 cards

Card naming convention: `{type}-{number}-{version}.webp` (e.g., `RW-A01a-PGa.webp`)

## 重要提示 / Important Notes

- `cardData.js` is auto-generated by `generate_list.js` - manual edits will be lost
- After adding new card images, run `node generate_list.js` to update data
- All development should be in Chinese (per project AI behavior standards)
